# IHM Face Detection Project

Projet réalisé dans le cadre du module :
Interfaces Homme-machine : Concepts, techniques et applications.

## Technologies utilisées
- Python
- Dlib
- OpenCV
- NumPy

## Fonctionnalités
- Détection de visage
- 68 facial landmarks
- Détection du clignement des yeux (Blink Detection)
